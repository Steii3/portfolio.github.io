Project Bowling

mise en situation

cette application permet de gerer les centres, clubs, joueur et compétitions de bowling, elle est écrite en C# avec une interface windows form, elle est relié a une base de données mysql

Logiciel utilisé Language de programmation Systeme d'opperation
Visual Studio Community 2019
phpmyAdmin
Wamp		 Csharp
SQL		 Windows

resultat

rechercher,modifier & supprimer :

chaque table de la base de données a un onglet ou il est possible de rechercher une information par colonne

A chaque changement sur la barre de recherche , une requete est envoyé a la base de données pour changer le contenu de l'affichage, cela permet d'avoir une interface interractive

il est possible de supprimer ou modifier des informations grace aux boutons du bas. Seul les tables spécifié dans le cahier des chage sont modifiable

créer :

il est possible de créer les differents tables avec la liste deroulante en haut a droite
