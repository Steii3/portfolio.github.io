Casse Briques

mise en situation

La crise sanitaire ayant été le facteur d'annulation de mon stage de 1ère année, mon professeur de formation m'a confié un projet a réaliser,

ce projet a pour but d'améliorer un Jeux casse-Brique crée en C# form,
Chaqu'un de ces modifications devait etre finalisé, :

  • Ajout d'une brique qui rétrécit la barre , ajout d'une brique qui annule tous les effets
  • Le joueur a 3 vies
  • Ajoutez une brique qui se casse en 3 fois
  • Tous les effets s’annulent au bout de 30 secondes
  • Le jeu possède plusieurs murs de briques (plusieurs niveaux)
  • Ajoutez une brique qui dédouble la boule
  • Créez un générateur de niveaux
Logiciel utilisé Language de programmation Systeme d'opperation
Visual Studio Community 2019 Csharp
XML serialization		 Windows

resultat

toutes les modifications ont été réalisé

interface de jeux :

listes des briques spéciales :

  • brique qui accelere la boule
  • brique qui multiplie la boule
  • brique qui se détruit en 3 coups
  • brique qui est indestructibles
  • brique qui annules tout les effets negatifs

interface de création de niveau :

Ce menu permet de :

  • créer un niveau en plusieurs parties tout en les modifiant individuellement
  • créer un niveau aléatoire avec un pourcentage de brique spéciale
  • placer et retirer chaque type de briques
  • charger un niveau existant et le modifier
  • Sauvegarder le niveau
